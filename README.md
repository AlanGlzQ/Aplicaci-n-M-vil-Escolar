# Aplicaci-n-M-vil-Escolar
Aplicación desarrollada en ReactNative como proyecto escolar/personal, implementando features como Loggin, mapa interactivo, directorio y video de la institución 

## Servicios externos

- Autenticación y datos académicos: `https://cuceimobile.space/campusCucei/auth.php`
  (API externa, no desarrollada por mí).
- Directorio de CUCEI: `https://cuceimobile.space/directorio.html`
  (página externa, solo se muestra embebida en un WebView).
- Sección de videos: contenido hosteado en GitHub Pages.
